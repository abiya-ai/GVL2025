# **App Name**: Vibe Coding Sessions

## Core Features:

- Homepage Navigation: Provide links to various coding workshop sessions from the homepage.
- Session Page: Each session has its dedicated page.
- Recording Link: Display a prominent link to the recording of each session using the [URL_TO_RECORDING].
- Slide Deck Link: Display a prominent link to the slide deck for each session using the [URL_TO_SLIDES].
- Lab Document Embed: Embed the lab document for each session using an iframe from [EMBED_URL_TO_LAB_DOC].

## Style Guidelines:

- Header Background: #FFFFFF (Pure White)
- Active Tab/Current Page Underline: #4285F4 (Google Blue)
- Inactive Tab Text: #3C4043 (Near-Black Gray)
- Tab Hover: A slightly lighter blue or a subtle box shadow.
- General Links (Recordings/Slides): #4285F4 (Google Blue)

- Body Background: #F8F9FA (Off-White)
- Embedded Document (e.g., <iframe> border): A very thin border in #F4B400 (Google Yellow) can frame the content without being distracting.
- Error messages and warnings: Use #FF0000 (Pure Red) for high-visibility alerts.
- Body and headline font: 'Inter' for a modern, clean, and readable look.
- Use a clean, simple layout with sufficient white space for easy viewing of content.
- Use clear and simple icons for navigation and key elements.
- Subtle hover effects on navigation links and buttons to provide visual feedback.
